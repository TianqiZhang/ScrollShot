Frame Capture & Zone Detection Pipeline

Step-by-Step Processing
1. CAPTURE
DXGI Desktop Duplication
GPU-accelerated screen capture of the selected rectangle.
Triggered on each scroll event. Background thread.
Falls back to BitBlt (GDI) on older systems.
2. ZONE SPLIT
Detect Fixed vs Scrolling Regions
Compare Frame N with Frame N-1 row-by-row (vertical) or column-by-column (horizontal).
Fixed rows/cols: pixel difference below threshold → part of the chrome.
Scrolling band: contiguous region where content has shifted.
Runs once on the first two frames to establish zones, then re-verified periodically.
Output: fixedTop, scrollBand, fixedBottom (or left/right for horizontal).
3. OVERLAP
Match Within the Scrolling Band Only
Extract the scrolling band from both frames.
Compute row/column hash signatures within the band.
Slide new frame's band against previous to find best overlap (minimize SAD).
No match found? Concatenate directly — never miss content.
100% match? Skip frame (content hasn't moved yet).
4. STITCH
Append Non-Overlapping Delta
Append only the new (non-overlapping) portion of the scrolling band to the segment list.
Fixed chrome is captured once and stored separately.
Final composition: fixedTop + all stitched scroll segments + fixedBottom.

Key Technical Decisions

Design Choices
Screen Capture API DXGI Desktop Duplication (primary), BitBlt/GDI (fallback). GPU-accelerated, captures at display refresh rate.
Zone Detection Row/column diff between first two frames. Rows below a pixel-difference threshold are "fixed". Contiguous changed rows form the "scrolling band". Re-verified every ~10 frames to handle dynamic chrome.
Direction Lock Determined by first scroll input. Wheel delta Y → vertical. Shift+wheel or trackpad X → horizontal. Locked for the session.
Frame Storage List of non-overlapping delta segments. Only composited at save time. Memory-bounded — we never store duplicate pixels.
Overlay Window Transparent, topmost WPF window. Click-through except for the Done button. Uses WS_EX_TRANSPARENT + WS_EX_TOOLWINDOW to avoid capturing itself.
