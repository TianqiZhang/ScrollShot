Component Responsibilities

What Each Layer Does
App Shell Entry point. Registers global hotkey (Win+Shift+S or configurable). Shows system tray icon with context menu. Small optional control window with "New Capture" button. Manages user settings (hotkey, save folder, etc).
Selection Overlay Transparent topmost WPF window spanning all monitors. Dims screen with semi-transparent black. Crosshair cursor. Click-drag to select rectangle. Passes input events (Enter, Scroll, Esc) to the appropriate handler. Hosts the live preview strip panel during scroll capture.
Capture Layer Wraps DXGI Desktop Duplication API. Captures a specified screen rectangle as a bitmap, cursor excluded. Falls back to BitBlt/GDI if DXGI unavailable. Runs on a dedicated thread. Provides raw frame bitmaps to consumers.
Scroll Engine Core logic. Zone detection: scans all 4 edges inward to find the scrolling rectangle. Overlap matching: row/column hashing + SAD within the scrolling band. Stitching: maintains ordered list of delta segments. Dedup: skips identical frames. Emits events for the live preview to consume.
Preview Editor Displays the stitched result. Timeline strip: visual representation of the long image — drag handles to trim head/tail, click to split and remove middle sections. Crop tool. Toggle to include/exclude fixed chrome. Save as PNG, copy to clipboard, or discard.

Key Cross-Cutting Concerns

Things That Span Components
Multi-Monitor Overlay spans all monitors. Capture coordinates are in virtual screen space. Selection can cross monitor boundaries.
DPI Awareness Per-monitor DPI-aware (v2). Capture uses physical pixels. UI scales correctly across mixed-DPI setups.
Memory Delta-only storage during capture. Segments composited at save time. For very long scrolls, segments can be flushed to temp files to bound memory.
Self-Exclusion Overlay uses WS_EX_TRANSPARENT + WS_EX_TOOLWINDOW. The preview strip and Done button use a separate layered window that is excluded from capture by SetWindowDisplayAffinity or by capturing before compositing the overlay.
